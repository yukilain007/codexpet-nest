# CodexPet Nest v0.1 Feature Matrix

## Status Definitions
- **Supported**: Fully implemented and tested.
- **Partial**: Implemented with known gaps or limitations.
- **Not Supported**: Not planned for v0.1.

## Core Features

| Feature | Status | Notes |
| --- | --- | --- |
| Nest Overlay Window | Supported | Transparent, always-on-top, follows pet. |
| Pet Position Tracking | Supported | Via read-only access to `.codex-global-state.json`. |
| Multi-monitor Support | Supported | Automatic coordinate conversion. |
| Clock Widget | Supported | Time and date display. |
| Countdown Widget | Supported | Persistent goal timer. |
| Pomodoro Widget | Supported | Focus/rest intervals with notifications. |
| Usage Indicator | Supported | Based on local `logs_2.sqlite` read-only parsing. |
| Settings Persistence | Supported | `settings.json` in App Support. |
| Online Pet Marketplace | Supported | Browse, download, SHA256 verify, install. |
| Local Pet Manager | Supported | Scan, list, open folder, uninstall app-managed pets. |
| Online Nest Marketplace | Supported | Browse, download, SHA256 verify, install. |
| Local Nest Manager | Supported | Install local ZIP, preview, switch active skin. |
| Nest Skin Applying | Supported | Real-time "Apply Now" from marketplace or local list. |
| Safe Extraction | Supported | `SafeZipReader` protects against traversal and symlinks. |

## Limitations & Gaps

| Feature | Status | Notes |
| --- | --- | --- |
| Auto-Switch Pet | Not Supported | Manual switch in Codex required after install. |
| Auto-Update | Partial | Manual download of new DMG required for v0.1. |
| Nest Editor | Not Supported | Planned for future creator tools. |
| Scripting/Plugins | Not Supported | Security decision: static assets only for v0.1. |
| Intel Mac Support | Partial | Targeted for Silicon; Intel binary may work but untested. |

## Known Limitations
- The nest overlay may lag slightly during rapid window movement due to file-polling latency (0.5s).
- Some custom Codex pets may have non-standard ID formats; v0.1 supports `custom:<id>` and built-ins.
- Uninstallation only removes files installed by the app; unmanaged files in the pet directory are left untouched for safety.

## Manual Test Checklist (Pre-release)
- [ ] `make clean && make` succeeds without warnings.
- [ ] App starts and shows in menu bar.
- [ ] Nest window follows Codex pet movement.
- [ ] Right-click menu functions (Toggle widgets, Open settings).
- [ ] Online Marketplace loads pet list and previews.
- [ ] Installing a pet from marketplace works and shows "Install Successful" guide.
- [ ] Installing a nest from marketplace and clicking "Apply Now" updates the skin.
- [ ] Pomodoro timer triggers a notification on completion.
- [ ] Usage indicator updates when Codex activity occurs.
- [ ] Uninstalling a pet removes it from `${CODEX_HOME}/pets/`.
